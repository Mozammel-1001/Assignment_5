Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM such that- getElementsByClassName, getElementById, getElementsByTagName, querySelector, querySelectorAll, getAttribute etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: This property gets or sets the HTML content inside an element. It includes the HTML tags inside the element. It doesn't include hidden elements.

innerText: This property gets or sets the visible text content of an element. It also doesn't include hidden elements.

textContent: This property gets or sets the text content of an element. It includes hidden elements.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique in JavaScript by using which one can attach a single event listener to a parent element rather than individual child elements. This allows the event listener to handle events for multiple child elements, even if they are dynamically added or removed from the DOM.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event that occurs on an element bubbles up from the target element to its root in the DOM tree. This means that when an event is triggered on an element, it is first captured by the target element, and then it bubbles up to the parent elements, and so on, until it reaches the root of the document

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, you can create, add, and remove elements dynamically using the DOM. An element can be added by using createElement() method. append() and appendChild() method allows to add an element. Finally remove() and removeChild() removes an element. There are many more methods to add and remove an element while these are the most commonly used methods.
